## 核心问题诊断

### 1. 规则引擎的局限性

当前 `generate_minutes.py` 使用改进版规则引擎：

```python
# 行动项判断逻辑
strong_action_patterns = [
    r'(负责).+?(完成|提交|准备|制定)',
    r'(完成|提交|准备|制定).+?(?:日前|前|截止)',
]
```

**问题**：
- 只能匹配关键词，无法理解语义
- 无法区分"承诺"和"任务分配"
- 无法识别隐含的行动项

### 2. 结论提取完全缺失

```python
# generate_minutes.py 中 conclusion 始终为空
conclusion=""
```

**原因**：无规则匹配结论，需 AI 语义理解

### 3. 风险点误判

```python
risk_keywords = ['风险', '问题', '不稳定', '延迟', '延期', ...]
```

**问题**：关键词匹配导致误报（"没问题"被误判为风险）
